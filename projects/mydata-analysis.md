# 📊 마이데이터 분석 및 기획

**기간**: 2023년 ~ 현재  
**역할**: 데이터 분석 및 서비스 기획 주도  
**기여도**: 기획/분석 전반 100% 담당

## 🔍 프로젝트 개요

은행 내 마이데이터 2.0 정책 전환에 따라, 사용자 중심의 금융정보 통합 시스템을 기획·분석했습니다.  
전송 범위 설정, 전체 금융자산 조회 기능, 본인정보 관리 등 주요 기능에 대한 정책 해석과 기술 설계를 직접 주도했습니다.

## ✅ 주요 기여

- 마이데이터 2.0 정책 분석 및 기획 정리
- 전체 금융자산 통합 조회 화면 및 백엔드 전송 범위 설계
- 전송 유효기간 관리 로직 설계
- 전환 대상 고객군 도출 및 예상 전송 패턴 분석

## 🛠 사용 기술 및 도구

- Python, SQL을 활용한 분석
- Docker 기반 분석 환경 구성
- Tableau를 통한 부서 협업용 시각화

## 📈 성과

- 10만+ 사용자 대상 전체자산 조회 구조 설계 완료
- 유관부서 대상 정책 설명 자료 및 샘플 분석 결과 제공
